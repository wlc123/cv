## 自检和缺陷矫正
在转场和运动过快的情况，插帧算法不可避免的出现failcase。我们的模型会同步输出像素级置信度，如下图颜色越深置信度越低。通过置信度可以识别转场，在此情况选择复制前帧。
![](./3.png)
对运动过快情况可以对局部模糊来提高主观感受，在视频播放时运动模糊不会引起主观不适。如下图。
